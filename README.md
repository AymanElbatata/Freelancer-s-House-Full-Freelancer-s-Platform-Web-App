📌 Overview

Freelancer House is a complete web application designed to help freelancers manage their daily workflow efficiently.

It centralizes client management, project tracking, and task organization in one clean and modern system.

Payments are handled externally based on the freelancer’s preferred method.

✨ Features

👤 Client Management
Add and manage client information
Track all projects per client
Maintain complete client history

📁 Project Management
Create and manage multiple projects
Set project status (In Progress / Completed / Cancelled)
Track overall progress

✅ Task Management
Break projects into smaller tasks
Assign deadlines
Track completion and productivity
